# TelaLoginSENAI Flutter

## Descrição do Projeto (Português)
Este projeto é uma atividade do curso técnico de Desenvolvimento de Sistemas no SENAI Jundiaí, na disciplina de Programação para Dispositivos Móveis com o professor Thiago.

O aplicativo apresenta um fluxo de autenticação em Flutter com telas de login, cadastro e recuperação de senha, seguindo a identidade visual do SENAI.

### Funcionalidades
- Tela de splash inicial com carregamento
- Tela de login com validação de e-mail (obrigatório conter `@`)
- Navegação para cadastro de novo usuário
- Tela de recuperação de senha com validação de e-mail
- Layout adaptado para dispositivo móvel com estilo SENAI

## Project Description (English)
This project is an assignment for the Systems Development technical course at SENAI Jundiaí, in the Mobile Device Programming subject taught by Professor Thiago.

The app implements an authentication flow in Flutter with login, registration, and password recovery screens, following SENAI visual style.

### Features
- Initial splash screen with loading indicator
- Login screen with email validation (must contain `@`)
- Navigation to user registration screen
- Password recovery screen with email validation
- Mobile-friendly layout with SENAI-inspired design

## Como Executar / How to Run
1. Instale o Flutter: https://docs.flutter.dev/get-started/install
2. Abra o projeto no VS Code ou Android Studio
3. Execute `flutter pub get`
4. Execute `flutter run`

## Estrutura do Projeto / Project Structure
- `lib/main.dart` - ponto de entrada da aplicação e tela de splash
- `lib/ui/login_page.dart` - tela de login
- `lib/ui/register_page.dart` - tela de cadastro
- `lib/ui/recover_password_page.dart` - tela de recuperação de senha
